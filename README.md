## ✨ Recursos

- **🎨 Interface Moderna** - Design glassmorphism com efeito de transparência e gradiente colorido
- **🔢 Operações Matemáticas** - Adição, subtração, multiplicação e divisão
- **⚡ JavaScript Vanilla** - Sem dependências externas, 100% funcional
- **📱 Responsivo** - Otimizado para desktop e mobile
- **⌨️ Controles Intuitivos** - Botão Limpar (C) e Backspace (←)
- **⚙️ Leve e Rápido** - Carregamento instantâneo

## 🎯 Como Usar

1. **Abra a calculadora** no seu navegador
2. **Clique nos números** para inserir valores
3. **Selecione uma operação** (+, −, ×, ÷)
4. **Pressione "="** para ver o resultado
5. **Use "C"** para limpar ou **"←"** para apagar

### Exemplos
```
15 + 7 = 22
100 ÷ 5 = 20
12 × 3 = 36
50 − 15 = 35
```

## 📦 Instalação

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)

### Passos

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/calculator-js.git
   cd calculator-js
   ```

2. **Abra o projeto:**
   - Duplo clique em `index.html`, ou
   - Use um servidor local:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js
     npx http-server
     ```

3. **Acesse em seu navegador:**
   ```
   http://localhost:8000
   ```
